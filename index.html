🔄 버튼으로 새 데이터를 받으려면 GitHub Personal Access Token이 필요해요. 한 번 입력하면 이 브라우저에만 저장되고, 다른 곳으로 전송되지 않아요.

📝 토큰 발급 방법 (펼치기)
  1. GitHub 우측 상단 프로필 → Settings
  2. 좌측 메뉴 맨 아래 Developer settings
  3. Personal access tokens → Fine-grained tokens
  4. Generate new token 클릭
  5. Token name: summerholiday-refresh (아무거나)
  6. Expiration: 원하는 기간 (90 days 권장)
  7. Repository access: Only select repositories → summerholiday 선택
  8. Permissions → Repository permissions → Actions: Read and write
  9. Generate token → 토큰 복사 (다시 못 봐요!)
  10. 아래 칸에 붙여넣고 저장
